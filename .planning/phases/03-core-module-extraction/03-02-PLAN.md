<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| `lua/story/bootstrap.lua` → `lua/core/keymaps.lua` | Global keymaps are now loaded through a dedicated module and must stay explicit. |
| `lua/core/keymaps.lua` → Neovim runtime | Incorrect mapping mode or rhs values would silently change interactive behavior. |
| `lua/core/keymaps.lua` ↔ `lua/plugins/*` | Core/global mappings must not absorb plugin-local mappings and blur ownership. |
